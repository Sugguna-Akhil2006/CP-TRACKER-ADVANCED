# 🚀 CRT Companion

### AI-Powered Competitive Programming Assistant for Campus Recruitment Training

> “Consistency beats talent — this tool makes sure you stay consistent.”

---

## 📌 Overview

**CRT Companion** is a smart web application designed to help students improve their competitive programming performance during Campus Recruitment Training (CRT).

It goes beyond simple tracking by providing **intelligent reminders, performance analytics, and AI-driven suggestions** to help users stay consistent and focused.

---

## 🎯 Problem Statement

Most students participating in CRT:

* ❌ Lack consistency in daily practice
* ❌ Don’t know what to practice next
* ❌ Fail to analyze mistakes after contests
* ❌ Lose motivation over time

**CRT Companion solves this by turning raw coding activity into structured, actionable insights.**

---

### 📊 Dashboard

* Overview of problems solved
* Current and longest streak

* CRT session reminders (weekly)

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/crt-companion.git
cd crt-companion
```

---

### 2. Setup Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
python app.py
```

---

### 3. Setup Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in backend:

```
OPENAI_API_KEY=your_api_key
MONGO_URI=your_mongodb_uri
```
